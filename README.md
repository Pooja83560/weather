# Weather Agent Chat Interface

This project is a simple chat-based web application that allows users to ask questions about weather conditions in different locations.  
The application sends the user's message to a weather agent API and displays the response in a chat interface.

The project is built using **React, TypeScript, and Vite**.

---

## Features

- Chat interface for asking weather-related questions  
- Real-time responses from the weather agent API  
- Message history shown in the chat window  
- Auto-scroll to the latest message  
- Basic error handling for network or API issues  
- Option to clear the chat conversation  

The interface is responsive and works on desktop and mobile devices.

---

## Technologies Used

- React 18
- TypeScript
- Vite
- CSS Modules
- Fetch API
